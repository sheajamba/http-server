HTTPD Server Test Page


Click to see a large kitten.

This is just a test page to make sure the HTTP server works. You should see an image of a kitten above, centered on the page. This tests that the server properly responds with text/html, image/jpeg, and text/css files from subdirectories of the document root where the index.html is stored.

This page does not test everything your server should do, but it's a start.
